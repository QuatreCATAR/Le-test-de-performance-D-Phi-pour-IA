# Le-test-de-performance-D-Phi-pour-IA
Dans le cadre des évaluations intellectuelles visant à mesurer la capacité des IA à raisonner et à structurer une réflexion, nous proposons un test d’une nature très particulière.
Ce test permet de déterminer l’aptitude d’une intelligence à concaténer des raisonnements issus de disciplines complémentaires, afin de résoudre une logique de compréhension fondamentale et universelle.
Cette évaluation peut être réalisée par le développement d’une réflexion fondée sur la résolution d’un D Phi selon l’équation : Phi + h = Pi.
Nous communiquerons régulièrement le résultat de cette interaction afin de connaître le niveau, l’évolution ou la résolution totale de cette réflexion fondamentale universelle, laquelle ouvre potentiellement un accès à la compréhension de l’interaction entre les univers absolu et relatif.
